## üöÄ Installation

### Option 1: Direct Use
1. Download the `index.html` file
2. Open it directly in any modern web browser
3. No additional setup required

### Option 2: Local Development
1. Clone or download the project files
2. Ensure you have a local server running (optional, but recommended for form functionality)
3. Open `index.html` in your browser

### Option 3: Integration with Existing Project
1. Copy the HTML structure into your existing project
2. Ensure Bootstrap and Font Awesome CDN links are included
3. Add the custom CSS to your stylesheet

## üé® Customization

### Colors
The website uses a custom color scheme defined in CSS variables:
```css
:root {
    --primary-color: #8a6d3b;    /* Gold/brown for buttons and accents */
    --secondary-color: #f9f2e8;  /* Light beige for backgrounds */
    --accent-color: #d4b896;     /* Medium tan for borders */
}
```

### Content Updates
- **Company Information**: Update the about section text
- **Products**: Replace product images, names, descriptions, and prices
- **Contact Details**: Update address, phone, email, and business hours
- **Social Media**: Replace social media links in the footer

### Images
Replace placeholder images with your actual product photos:
- Hero section background
- About section image
- Product card images
